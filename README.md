# Motivation

假设存在一个字幕组或者汉化组，把完成的合法，合规作品传到适当网站，但因为莫须有的原因被审核删除，那么就需要一个更好的分享位置，最好是权力掌握在自己手里的地方。OneSS便出现了

这是LSW计划的一部分，让OneSS更像OSS一样的使用
